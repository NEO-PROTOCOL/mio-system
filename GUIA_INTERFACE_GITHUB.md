## 🛠 Interagindo com Issues

1.  **Assign yourself:** Sempre assuma a autoria das suas tarefas.
2.  **Labels:** Use `identidade`, `core-stack` e `high-priority`.
3.  **Linked PRs:** Vincule toda Issue a um Pull Request para rastreabilidade total.
